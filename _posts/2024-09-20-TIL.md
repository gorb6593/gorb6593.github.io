``` 

AWS 에서 EC2 인스턴스를 만들면 기본시간은 UTC이다 

시간을 변경해야 한다 

date -> 지금 시간 확인

sudo vim /etc/sysconfig/clock

ZONE="Asia/Seoul"
KST=True

cat /etc/localtime
rm /etc/localtime
ln -s /usr/share/zoneinfo/Asia/Seoul /etc/localtime

역시 쉽지않다..
```
